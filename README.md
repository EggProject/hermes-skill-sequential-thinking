## 💡 What is this?

`sequential-thinking` is a [Hermes Agent skill](https://hermes-agent.dev)
that exposes the Sequential Thinking MCP server as a first-class Hermes
tool. It is for complex problems where reasoning cannot be done in a
single shot — multi-stage analysis, design planning, problem
decomposition, root-cause debugging, and tasks with initially unclear
scope.

Under the hood the skill calls
`mcp_sequential_thinking_sequentialthinking` — the canonical Hermes
MCP tool name produced by registering the official Sequential
Thinking MCP server under the stable name `sequential-thinking`
(hyphens are converted to underscores by the MCP tool registry).

The repository is a **hand-ported** version of the upstream skill at
[`mrgoonie/claudekit-skills`](https://github.com/mrgoonie/claudekit-skills).
It is **not** generated by Anthropic's `skill-creator` — the layout
is rewritten to match Hermes' skill hub/tap conventions, and the
`SKILL.md` frontmatter is tuned for Hermes discovery (tags, related
skills, MCP tool name).

## 🎯 Why this skill?

| Pain point | How this skill solves it |
|------------|--------------------------|
| "I jump to a solution before understanding the problem." | Forces a sequential `thought` chain with explicit `thoughtNumber` / `totalThoughts` pacing. |
| "I have to backtrack, but my reasoning has no memory of where I went wrong." | `isRevision` + `revisesThought` keep corrections anchored to the original step. |
| "Multiple approaches look viable — I want to compare them, not commit blindly." | `branchFromThought` + `branchId` open parallel reasoning paths from any decision point. |
| "My initial scope estimate is always wrong." | `totalThoughts` is freely adjustable mid-run — progress visibility, not a contract. |
| "The Sequential Thinking MCP tool is hard to discover in Hermes." | Stable `sequential-thinking` server name → predictable `mcp_sequential_thinking_sequentialthinking` tool name. |
| "I want to install it as a Hermes tap, not copy files by hand." | `skills/sequential-thinking/` layout is tap-ready; one command installs the full bundle. |

## ✨ Features

- **Hermes-native MCP tool name** — `mcp_sequential_thinking_sequentialthinking`, derived from the stable `sequential-thinking` server name
- **Two reference files** — `references/advanced.md` (revision and branching patterns) and `references/examples.md` (four real-world workflows)
- **Iterative reasoning** — sequential `thought` steps with explicit `thoughtNumber` / `totalThoughts` pacing
- **Revision tracking** — `isRevision` / `revisesThought` to correct earlier conclusions without losing the chain
- **Branch exploration** — `branchFromThought` / `branchId` to walk alternative paths in parallel
- **Dynamic scope adjustment** — change `totalThoughts` freely as understanding evolves
- **Hand-ported for Hermes** — tuned SKILL.md frontmatter, tap-friendly layout, MIT-licensed upstream

## 🚀 Quick start

1. **Install the skill** — see [📦 Installation](docs/installation.md).
2. **Configure the Sequential Thinking MCP server** — same page.
3. **Verify** — same page. In a fresh Hermes session the tool is
   `mcp_sequential_thinking_sequentialthinking`.

That's it — start the next complex problem with
*"think this through step by step"* and the skill takes over.
